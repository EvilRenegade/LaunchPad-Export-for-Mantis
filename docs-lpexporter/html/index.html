LaunchPad Export for Mantis

LaunchPad Export for Mantis is a plugin which adds an additional export option on View Issues, allowing an administrator to export the selected bugs to LaunchPad's import XML schema.

The exported file will be put into the uploads folder and can be linked or downloaded from there.

As a secondary component, if the administrator configures the project's LaunchPad name, the plugin will display a bright "This bug has moved." notification, with a link to the bug's location on LaunchPad.

Configure your LaunchPad project name

To configure your LaunchPad project name to enable the "bug moved" notice after your bugs have been imported into production, proceed as follows:

  • Go to Manage -> Manage Configuration -> Configuration Report
  • Under Set Configuration Option add:
    Username: All Users
    Project name: Your project
    Configuration Option: plugin_LaunchPadExport_launchpad_project
    Type: string
    Value: Your project name on LaunchPad.
  • Click the Set Configuration Option button.

Missing features

This exporter was designed and tested on a single instance of Mantis, for a single export venture. As a result, differing configurations may lead to trouble and/or the need to patch.
One example would be uploads conversion: The test Mantis had the default "uploads in database" configuration, and that's what the code works with. If your uploads lie on disk at the moment, you will have to alter the code to read the file contents from there instead.
If you do patch the exporter to work with a different configuration, please contribute the code back to the project so others can benefit.

Information

The LaunchPad Export for Mantis plugin was written by Charly "Renegade" Kiendl in November 2011 for Mantis 1.2.5. It should work for all Mantises of the 1.2.x series, but no other version has been tested so far. Patches are welcome.
