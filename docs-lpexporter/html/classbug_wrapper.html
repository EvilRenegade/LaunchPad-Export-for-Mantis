LaunchPad Export for Mantis 2.0

getFullDescription ( ) [private]

Gets the full description text of this bug.

Mantis is finer-grained than LaunchPad; it offers the additional fields of "Steps to Reproduce" and "Additional Information". In order to transfer all information, we have to not just take the plain description, but munge the additional fields into it.

getLaunchPadImportance ( ) [private]

Deduces the LaunchPad importance for this bug.

Mantis is finer-grained than LaunchPad; it has both a priority and a severity; Launchpad only has importance. This function takes Mantis's two indicators and distills them down to one LaunchPad importance.

See also:
https://help.launchpad.net/Bugs/Statuses/External#Mantis
Returns:
The LaunchPad importance for this bug.
getLaunchPadStatus ( ) [private]

Deduces the LaunchPad status for this bug.

Mantis is far finer-grained than LaunchPad; it has a status, a resolution and a reproducibility; Launchpad only has a status. This function takes Mantis's three indicators and distills them down to one LaunchPad status.

See also:
https://help.launchpad.net/Bugs/Statuses/External#Mantis
Returns:
The LaunchPad status for this bug.
getMilestone ( ) [private]

Deduces the LaunchPad milestone for this bug.

Mantis is finer-grained than LaunchPad; it has both a target version and fixed-in-version-field; Launchpad only has one milestone. This function takes Mantis's two indicators and chooses one to be the LaunchPad milestone.

Returns:
The LaunchPad milestone for this bug.

getTags ( $  pDescription = NULL) [private]

Returns the tags for this bug.

Gets the tags associated with this bug in Mantis, and returns those in any case.
Additionally, if an argument is given, it'll extract the ten most-used words of three letters or longer from the text given and use those as tags as well. (This is to generate topical tags for bugs that haven't been tagged in the past.)

Parameters:
$pDescriptionA text to extract tags from, in our case the full description.
Returns:
An array of strings, the extracted tags.

tagHelper ( &$  pValue,
$  pKey 
) [private]

This is a callback-function for array_walk() to reduce all potential tags to lpnames.

This is necessary because otherwise, duplications and stop words can occur in the final output despite the fact that getTags should have filtered them out.
e.g. Armor=Crap and armor-crap would be equal in the eyes of lpname/Launchpad.

Parameters:
$pValueReference to the current array value.
$pKeyCurrent array key.
Returns:
Nothing, but writes to $pArray.
